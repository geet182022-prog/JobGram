# 🚀 JobGram – Premium MERN Job Portal

JobGram is a full-stack role-based job portal built using the MERN stack.  
It supports Candidates, Recruiters, and Admin workflows with authentication, resume upload, job approvals, and application tracking.

---

## 🌟 Features

### 👤 Authentication
- JWT-based login & registration
- Role-based access (Candidate / Recruiter / Admin)
- Protected routes

---

### 🧑‍💼 Candidate Features
- Upload Resume (PDF mandatory before applying)
- Apply for Jobs
- Withdraw Applications
- Save Jobs
- View Application Status (Pending / Selected / Rejected)
- Dashboard with application stats

---

### 🏢 Recruiter Features
- Post Job
- Edit/Delete Job
- View Applicants
- Download Resume
- Update Application Status
- View Applicant Count
- Approval Status Badge (Approved / Pending)

---

### 🛡 Admin Features
- Approve Jobs
- Reject Jobs
- Dashboard Analytics
- System Stats

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer (Resume Upload)
